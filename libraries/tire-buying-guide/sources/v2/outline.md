# 轮胎选购指南 V2 目录

- 第一章：轮胎的基本结构和参数
  - 轮胎的基本结构：胎面、胎肩、胎侧、钢丝带束层、胎体、内衬层、胎圈
  - 轮胎参数：胎宽、扁平比、子午线结构、轮毂尺寸、载重指数、速度级别
- 第二章：轮胎分类
  - 场景分类：夏季胎、四季胎、冬季胎、AT/MT 越野胎
  - 功能分类：节能/耐磨胎、性能胎、新能源专用胎、静音胎、自修复胎、防爆胎
- 第三章：各大主流品牌轮胎产品
  - 第一梯队：米其林、马牌、倍耐力
  - 第二梯队：普利司通、固特异、邓禄普、韩泰
  - 第三梯队：朝阳、赛轮、佳通
- 第四章：轮胎常见误区和企业闭口不谈的消费黑洞
  - 常见误区：新胎装前后、只换一条、花纹深、混用品牌、胎侧鼓包、动平衡、四季胎冬季、夏季降胎压
  - 消费黑洞：静音棉、自修复/防爆、低滚阻、耐磨指数、大厂子品牌、OE 认证标
- 第五章：轮胎选购决策
